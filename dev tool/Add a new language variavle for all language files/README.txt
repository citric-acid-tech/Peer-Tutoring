用途
在所有的translation_lang.php后面追加新的一行变量，避免部分体力劳动，节省时间。
--------------------------------------------------------------------------------
第一次启动：
--------------------------------------------------------------------------------
1. 输入set
2. 输入language文件夹的路径，例如C:\AppServ\Peer-Tutoring\src\application\language
3. 同一目录会产生一个设置文件保存这个路径以便下次使用
--------------------------------------------------------------------------------
insert指令
1. 输入insert
2. 输入键值，如：hint_cancel
3. 输入内容，如：Are you sure to cancel this?
所有的语言文件底部都会插入: $lang['hint_cancel'] = 'Are you sure to cancel this?';
--------------------------------------------------------------------------------
delete指令
1. 输入delet
2. 输入键值，如：hint_cancel
所有的语言文件中，包含$lang['hint_cancel']的行都会被删除
--------------------------------------------------------------------------------
